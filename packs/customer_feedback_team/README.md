# Customer Feedback Intelligence Team

Analyze feedback CSVs into complaints, severity, and executive recommendations.

## How to use

1. Start AgentForge Studio.
2. Open the Template Gallery.
3. Select this pack.
4. Upload the sample input or paste your own task.
5. Run the workflow and inspect traces.

## Sample task

```text
Analyze customer feedback, group major complaints, classify severity, and generate an executive report.
```

## Included files

- `pack.yaml` — portable Agent Pack definition
- `eval_cases.json` — rule-based evaluation cases
- `sample_input.md` — sample prompt/input
